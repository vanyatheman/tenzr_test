Решение данной задачи состоит в следующем:

1. В классе DataImporter происходит импорт данных из JSON файла в БД SQLite3
при помощи библиотек sqlite3 и json. Создается таблица data в БД с помощью
запроса 'CREATE TABLE IF NOT EXISTS data' и вставка JSOn данных в нее: 
'INSERT OR IGNORE INTO data'

2. В классе StaffFetcher происходит обработка иерархической структуры с помощью
рекурсивного запроса. Он использует временную таблицу Rec (WITH RECURSIVE Rec).
В этой таблице он выбирает все данные из таблицы data, где ParentId равен id
офиса, который мы получили в прошлом рекусрсивном запросе. Это начальная часть
рекурсии - уровень равен 0. Затем с помощью UNION ALL он объединяет с этой
начальной частью данные таблицы, где ParentId равен id из предыдущего уровня.
При этом уровень повышается на 1. Это обеспечивает рекурсию. После заполнения
запрос отдает нам сотрудников из офиса (тип 3).
Такая же логика нахождения офиса. Выбраем сотрудника по id. Только наоборот -
подинмаемся вверх по ParentId до офиса (тип 1).
